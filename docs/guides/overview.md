# Guides Overview

The Guides section is for **end users and operators** of CDT. Each page is a task-oriented walkthrough — pick the goal you have in mind, follow the steps, and confirm the result before moving on.

If you have not signed in yet, start with the [Quickstart](../getting-started/quickstart.mdx). It threads the most important guides together into a single 30-minute path.

## Format

Every guide follows the same structure:

- **Goal** — what you are trying to achieve.
- **Prerequisites** — what you need before starting.
- **Steps** — numbered, with the expected screen response after each.
- **Result** — what you should see when finished.
- **Related** — pages that go deeper or branch off.

Skim the goal and result first to confirm a guide is the right one before reading the steps.

## Conventions

- Keystrokes use the format `Ctrl + S` (Windows/Linux) or `Cmd + S` (macOS).
- "Click" assumes a mouse or trackpad. Touch users can substitute "tap."
- Code blocks are copy-pasteable — placeholders in `<angle brackets>`.
