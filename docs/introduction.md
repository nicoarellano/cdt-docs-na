# Introduction

The Collab Digital Twins platform (CDT) is a web-based platform for the visualization of and interaction with multi-scale geospatial information systems, open data, and BIM models.

## Overview

CDT bridges BIM and GIS using open standards and free, open-source technologies, enabling stakeholders to visualize and analyze data directly in the browser.

## Core Values

- **Open Standards** — Eliminate proprietary barriers and promote interoperability through open standards
- **Community Driven** — Build with and for the community
- **Transparent** — Foster trust and collaborative development through open source code
- **Imagination** — Challenge current boundaries that limit what is possible with open source digital technologies
- **Multidisciplinary** — Where we are today is evidence that we are stronger and smarter together
- **Accessible** — Empower everyone with intuitive, user-friendly tools that work directly in the browser
